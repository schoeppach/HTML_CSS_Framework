Webentwicklung Webdesign

Als Webentwicklung bezeichnet man die Softwareentwicklung von Webanwendungen, Webservices oder anderer komplexer Websites. Die in der Webentwicklung genutzten Technologien veränderten sich parallel zur Entwicklung des Internets, einige der wichtigsten Verträter sind...

HTML

Die Hypertext Markup Language ist eine textbasierte Auszeichnungssprache zur Strukturierung elektronischer Dokumente wie Texte mit Hyperlinks, Bildern und anderen Inhalten. HTML-Dokumente sind die Grundlage des World Wide Web und werden von Webbrowsern dargestellt.

CSS

Cascading Style Sheets ist eine Stylesheet-Sprache für elektronische Dokumente und zusammen mit HTML und JavaScript eine der Kernsprachen des World Wide Webs

PHP

PHP (rekursives Akronym und Backronym für „PHP: Hypertext Preprocessor“) ist eine Skriptsprache mit einer an C und Perl angelehnten Syntax, die hauptsächlich zur Erstellung dynamischer Webseiten oder Webanwendungen verwendet wird.
